## üöÄ Getting Started

### Prerequisites
- Node.js 18.18.0 or higher
- npm or pnpm

### Installation
```bash
# Clone the repository
git clone <your-repo-url>
cd blog-starter-app

# Install dependencies
npm install
# or
pnpm install

# Start development server
npm run dev
# or
pnpm dev
```

### Build for Production
```bash
npm run build
npm start
```
